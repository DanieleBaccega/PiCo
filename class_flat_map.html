Detailed Description

template<typename In, typename Out>
class FlatMap< In, Out >

Defines an operator performing a FlatMap, taking in input one element from the input source and producing zero, one or more elements in output. The FlatMap kernel is defined by the user and can be a lambda function, a functor or a function.

It implements a data-parallel operator that ignores any kind of grouping or windowing.

The kernel is applied independently to all the elements of the collection (either bounded or unbounded).
