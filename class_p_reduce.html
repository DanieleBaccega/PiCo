Detailed Description

template<typename In>
class PReduce< In >

Defines a PReduce operator performing a tree reduce function on partitioned input (i.e. reduce by key). The reduce kernel is defined by the user and can be a lambda function, a functor or a function.

The reduce kernel function operates on windows and/or groups if defined on the Pipe.

It implements a tree reduce operator where input and output value are the same.
