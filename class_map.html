Detailed Description

template<typename In, typename Out>
class Map< In, Out >

Defines an operator performing a Map function, taking in input one element from the input source and producing one in output. The Map kernel is defined by the user and can be a lambda function, a functor or a function.

It implements a data-parallel operator that ignores any kind of grouping or windowing.

The kernel is applied independently to all the elements of the collection (either bounded or unbounded).
